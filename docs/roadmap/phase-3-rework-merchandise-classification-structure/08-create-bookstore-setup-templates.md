
# Create bookstore setup templates for classification defaults

## Summary

Create starter templates so stores do not have to build merchandise classes, category schemes, and mappings from scratch.

## Why

The classification model is powerful but could overwhelm users if exposed as a blank setup process. Stores should start from practical bookstore templates and adjust only what they need.

## Suggested templates

- Simple Bookstore
- ABA-style Bookstore
- Bookstore + Cafe
- Bookstore + Events
- Used Bookstore
- General Retail / Sidelines Heavy

## Template contents

Each template may include:

- merchandise classes
- starter category schemes
- category nodes
- default display location examples
- default accounting mappings
- default tax/pricing assumptions

## Acceptance criteria

- At least one starter template exists.
- Template can seed merchandise classes.
- Template can seed one category scheme and starter nodes.
- Template can seed basic accounting mappings.
- Existing setup flow can apply template defaults.
