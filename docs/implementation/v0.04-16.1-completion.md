## Notes

* `metadata_sections` Turbo Frame path remains as a temporary compatibility/debug endpoint and now uses `MetadataPreviewParams` instead of raw `assign_attributes`.
* Creators/subjects preview keeps nested `catalog-item-form`; Product form root no longer couples visibility to it.
* BISAC/genre canonical hidden inputs live in `data-product-canonical-inputs` regions; picker UI is separate.
* Hidden/disabled fields are absent from submit; sanitizer treats absence as not-submitted (does not clear DB values on edit).
* Legacy `workflow: catalog_linked|non_catalog` remain as assist aliases; primary Add Product path is unified.
* Product SKU is not editable on the normal Product form; identifiers are managed separately.
