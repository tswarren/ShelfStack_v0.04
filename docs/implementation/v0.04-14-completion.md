## UX contract (preserved)

* Page header: secondary actions → primary (primary rightmost)
* Form footer: primary submit left, tertiary Cancel right
* Destructive lifecycle actions in danger zone (not page header)
* Table/link actions: `variant: :link` where appropriate
* Do not rely on `:first-child` CSS for button hierarchy
